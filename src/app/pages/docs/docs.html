DevOops Website Documentation

Overview

This project is a collaborative AI-powered code review and task assignment platform that allows teams to streamline development workflows through GitHub integration and intelligent developer-task matching.

Key features

  1. User Authentication

    Sign up and log in with role-based access control and secure sessions.

  2. Workspaces

    Create multiple workspaces; creators become workspace admins and manage members.

  3. Developer Management

    Admins invite developers; developers connect GitHub for review integration.

  4. GitHub Integration for Code Review

    Submit branch + token; AI analyzes code and returns score, feedback and suggestions.

  5. AI-Powered Task Assignment

    Admins send tasks to the AI which recommends the most suitable developer based on history.

  6. Scoring & Performance Tracking

    Every review produces a score saved in developer profile for future matching and metrics.

Get started

  1. Get Your GitHub Token

    1. 1- Open GitHub Website

      Go to GitHub and log in.

    2. 2- Navigate to Settings

      Click on your profile picture in the top right corner, then select Settings from the dropdown menu.

    3. 3- Generate a Personal Access Token

      Navigate to Developer settings â†’ Personal access tokens. Click on Generate new token.

    4. 4- Configure Token Permissions

      Give your token a descriptive name [User Agent], set an expiration date, and select the necessary scopes (e.g., repo for full control of private repositories).

    5. Note: Copy Your Token

      After generating the token, copy it immediately. You won't be able to see it again.

How To Use The Website

Details on the steps of using the website

Register and Login

To use the website, you need to register and log in. Click on the "Register" button to create an account, then log in with your credentials,Or you can use the "Login" button if you already have an account.

After registering, you can log in using your email and password.

Home Page

Once logged in, you will be directed to the home page. Here, you can see your workspaces, create new ones, and manage your tasks. The home page provides an overview of your recent activities and notifications.

Profile

You can access your profile by clicking on profile button in the sidebar. In your profile, you can update your personal information,and view your performance scores.This feature allows you to track your progress and see how you are performing in the workspace.

Create Workspace

You can create a new workspace by clicking on the "Create Workspace" button. Enter the workspace name and description, then click "Create".Once created, you can manage the workspace settings, invite members, and assign tasks.

When creating the workspace, you will be prompted to enter a spicific data. The workspace creator will automatically become the admin of the workspace.

You should enter the GitHub token and user Agent you generated earlier in the "GitHub Token" field. This token is used to integrate your GitHub.Also,The owner name is the name of the user who created the workspace. The repo name is the name of the repository you want to connect to the workspace.

Admin Role

As an admin, you have the ability to manage the workspace, invite developers, and assign tasks. You can view the list of members, their roles, and their performance scores.

You can invite developers by clicking on the "Add New Member" button. Enter the developer's email address, and they will receive an invitation to join the workspace. Once they accept, they will be able to access the workspace.

Note: The Branch Name Is The User's Github Branch Name

After inviting developers, you can assign tasks to them. Click on the "Assign Task" button, select the developer, and enter the task details. The AI will analyze the developer's performance and suggest the best match for the task.

Developer Role

As a developer, you can view the tasks assigned to you, submit code for review, and check your performance scores. You can also see the feedback and suggestions provided by the AI after each code review.

To submit code for Build Check, click on the "Check Status" button. The AI will analyze your code and Tell You The Code Build Status.

The Response Will Be The Task Score, Feedback and Suggestion

To submit code for review,Choose the task you want to review, click on the "Submit Code" button.The AI will analyze your code and provide feedback.
